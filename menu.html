Importação do Cadastro de clientes do Contador.

Através da instrução abaixo o usuário aprender a importar os clientes que estão na Contabilidade para o Conversor5.
**Exclusivo para clientes que utiliza o sistema local.

Execute conforme ordem:
1- Faça configuração do ODBC para que consiga importar os dados direto das Soluções Domínio. Saiba como.
2- Dentro do Conversor acesse o ícone “clientes” na barra lateral esquerda, depois clique em GERENCIAR CLIENTES;
3- Na tela aberta, clique em IMPORTAR para importar os dados do Domínio.
